- cover

- figures, tables
- advanced kindle-compatible layout

How to build:
1. Get KindleGen.exe
2. Put KindleGen and sources with *.opf file in C:\
2. Run in command prompt
"C:\KindleGen.exe" "C:\CLR via CSharp.opf" -o2
o2 - compress flag (o0 - low, o2 - high)
